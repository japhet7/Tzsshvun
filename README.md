# TZSSHVUN V1 - Vulnerability Assessment Tool

**TZSSHVUN V1** is a professional ethical hacking tool designed to identify vulnerabilities in networks, operating systems, and web services. It automates the discovery of CVEs and generates a professional Word (.docx) report.

## Features
- **Network Scanning:** Identifies open ports and services.
- **Vulnerability Detection:** Uses Nmap Scripting Engine (NSE) to find known exploits.
- **Evidence-Based Reporting:** Generates a professional .docx report with raw evidence snippets.
- **Cross-Platform:** Runs on Windows, Debian, Ubuntu, and Kali Linux.
